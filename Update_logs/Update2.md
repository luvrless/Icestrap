# **Title:** **Icestrap Update Release (v1.0.5)**

**Text:**  
Icestrap v1.0.5 — a small maintenance release with improvements and fixes.

- Discord Rich Presence (RPC) fixed and is now working.
- Added and fixed Avatar Viewer in the Account tab.
- Improved the updater to retrieve the latest global release version, with a safe auto-increment fallback for reliable update detection.
- Playtime counter now updates live in the UI.

Thank you for using Icestrap! We appreciate your continued support and will continue improving the application with future updates and new features.

> **Date:** July 1, 2026
